# Constructora Clemente

Sitio web para un arquitecto con constructora propia. La experiencia presenta la propuesta de valor, proyectos, proceso, materiales, visualizaciones y contacto comercial.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Enfoque

- Arquitectura contemporánea con ejecución construible.
- Portafolio visual para transmitir criterio y confianza.
- Flujo de navegación de hero a proyectos, proceso, materiales y consulta.
